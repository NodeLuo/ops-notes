
1、Linux目录的作用

```bash
Linux下的一级目录的作用： (\目录下)

	bin        二进制系统命令所在的目录
	
	boot       系统内核的位置，引导程序运行
	
	dev        硬件设备所在的位置
	
	etc        配置文件所在的目录
	
	home       普通用户的家目录
	
	mnt        临时挂载点
	
	opt        第三方程序默认安装的位置
	
	proc       内存映射文件，查看硬件信息、cpu和内存
	
	root       管理员的家目录
	
	sbin       超级管理员命令所在的位置
	
	tmp        临时目录存放的位置，回收站（垃圾文件移动到这里替代删除）
	
	usr        用户程序文件所在的位置
	
	var        日志文件所在的位置
```

2、系统命令

```bash
1.cp 复制文件
	cp file dir
	cp file newfile
	cp file1 file2 dir
	cp -r dir1 dir2 递归复制目录
	\ 撬棍符号 -- 取消别名作用
		cp别名：cp -i 覆盖提示
		\cp file1 /dir/file1  不提示覆盖

2.mv 移动文件或目录
	mv file dir
	mv dir dir
```
