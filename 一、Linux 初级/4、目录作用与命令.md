
1、Linux目录的作用

```bash
Linux下的一级目录的作用： (\目录下)

	bin        二进制系统命令所在的目录
	
	boot       系统内核的位置，引导程序运行
	
	dev        硬件设备所在的位置
	
	etc        配置文件所在的目录
	
	home       普通用户的家目录
	
	mnt        临时挂载点
	
	opt        第三方程序默认安装的位置
	
	proc       内存映射文件，查看硬件信息、cpu和内存
	
	root       管理员的家目录
	
	sbin       超级管理员命令所在的位置
	
	tmp        临时目录存放的位置，回收站（垃圾文件移动到这里替代删除）
	
	usr        用户程序文件所在的位置
	
	var        日志文件所在的位置
```

2、挂载流程

```bash
1.虚拟机DVD中必须插入ISO镜像文件

2.创建一个挂载点，或者用/mnt 系统临时挂载点，dev/cdrom需要一个挂载点作为入口

3.挂载操作
	mount /dev/cdrom /mnt
 	
4.进入/mnt，查看镜像内容

5.查看磁盘信息
	df -h	

6.卸载挂载点
	umount /mnt
```

3、系统命令

```bash
1.cp 复制文件
	cp file dir
	cp file newfile
	cp file1 file2 dir
	cp -r dir1 dir2 递归复制目录
	\ 撬棍符号 -- 取消别名作用
		cp别名：cp -i 覆盖提示
		\cp file1 /dir/file1  不提示覆盖

2.mv 移动文件或目录
	mv file dir
	mv dir dir
	mv file1 file2 改名

3.rm 删除文件或目录
	rm file
	rm file1 file2
	rm -f 强制删除文件不提示
	rm -rf 递归强制删除目录不提示
	
	rm -rf * 会自动跳过隐藏文件，不会删除隐藏文件 （删除当前目录下的所有文件）

4.vim 文件编辑器
	编辑流程：
	a.vim file  进入视图模式
	b.按 a或i    进入编辑模式
	c.按 Esc    返回视图模式
	d.输入:wq    保存并退出

5.echo 输出字符串
	echo 字符串
	echo 字符串 > file 将内容保存到文件中
	输出重定向：
	a.>    覆盖
	b.>>   追加
	c.2>   错误内容覆盖（只接受错误）
	d.2>>  错误内容追加（只接受错误）
	
	序列：
	{0..10} {a..z}
	{0..10..2} {a..z..2} 间隔为2
	a{1..10}b 拼接
	
	示例
	[root@breadbomb ~]# echo {1..10..2}
	1 3 5 7 9
	
	[root@breadbomb ~]# echo a{1..10..2}b
	a1b a3b a5b a7b a9b
```
