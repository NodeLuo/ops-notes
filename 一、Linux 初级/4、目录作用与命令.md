
1、Linux目录的作用

```bash
Linux下的一级目录的作用： (\目录下)

	bin        二进制系统命令所在的目录
	
	boot       系统内核的位置，引导程序运行
	
	dev        硬件设备所在的位置
	
	etc        配置文件所在的目录
	
	home       普通用户的家目录
	
	mnt        临时挂载点
	
	opt        第三方程序默认安装的位置
	
	proc       内存映射文件，查看硬件信息、cpu和内存
	
	root       管理员的家目录
	
	sbin       超级管理员命令所在的位置
	
	tmp        临时目录存放的位置，回收站（垃圾文件移动到这里替代删除）
	
	usr        用户程序文件所在的位置
	
	var        日志文件所在的位置
```
