
1、特殊符号

```bash
1.<< 标准输入重定向
	示例：
		#追加文件内容而不使用vim
		cat >> a.txt << EOF （EOF 可以自定义，结束符）
		
		[root@breadbomb ~]# cat >> a.txt << EOF
        > hello
        > 你好
        > EOF
        [root@breadbomb ~]# cat a.txt
		hello
		你好
		
		#输出到屏幕
		cat << EOF
		
		[root@breadbomb ~]# cat << eof
        > hello
        > 你好
        > back
        > eof
		hello
		你好
		back

2. '' 和 ""
	单引号不会识别特殊内容（正则可以）
	echo '$LANG' ----> $LANG
	双引号会识别特殊内容
	echo "$LANG" ----> en-US.UTF-8
```
