
1、特殊符号

```bash
1.<< 标准输入重定向
	示例：
		#追加文件内容而不使用vim
		cat >> a.txt << EOF （EOF 可以自定义，结束符）
		
		[root@breadbomb ~]# cat >> a.txt << EOF
        > hello
        > 你好
        > EOF
        [root@breadbomb ~]# cat a.txt
		hello
		你好
		
		#输出到屏幕
		cat << EOF
		
		[root@breadbomb ~]# cat << eof
        > hello
        > 你好
        > back
        > eof
		hello
		你好
		back

2. '' 和 ""
	单引号不会识别特殊内容（正则可以）
	echo '$LANG' ----> $LANG
	双引号会识别特殊内容
	echo "$LANG" ----> en-US.UTF-8
```

2、正则表达式

```bash
三剑客：grep sed awk

1.^ 以...开头

[root@breadbomb ~]# grep '^h' a.txt
hello

[root@breadbomb ~]# grep '^[h你]' a.txt
hello
你好

2.$ 以...结尾

[root@breadbomb ~]grep 'o$' a.txt
hello

a.结尾有时候看不到空格：
	cat -A 查看每一行以什么结尾
	（1）^I 表示tab结尾
	（2）^$ 表示当前行为空行

b.$在前面表示变量，在双引号中会解析变量的值
	[root@breadbomb ~]# grep '$LANG' a.txt
	
	[root@breadbomb ~]# grep "$LANG" a.txt
	en_US.UTF-8


3. .（点）表示任意一个字符

a.任意一个字符过滤不会匹配到空行
	[root@breadbomb ~]# grep '.' a.txt
	hello 
	world	
	en_US.UTF-8




```
