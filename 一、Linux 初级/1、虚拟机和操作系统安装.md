- 配置和安装操作系统

```bash
1.将 centos7镜像 放入 虚拟机光驱

步骤：
	a.右键虚拟机
	b.点击CD/DVD
	c.选择使用ISO镜像文件，选择镜像文件地址

2.启动虚拟机

3.修改网卡名称为eth0

步骤：
	a.初始界面，移动到install centos 7，按tab键
	b.输入空格 输入biosdevname=0 net.ifnames=0

4.选择语言 English

5.安装配置界面，选择DATE & TIME，选择当前时区，点击done

6.点击language support，添加中文

7.点击software selection,左边选minimal install,右边除了security tools 和 smart card support 全选

8.点击installation destination，底部点击I will configure partitioning
	a.选择standard partition
	b.新建分区 /boot 200m
	c.新建分区 /swap 2048m
	d.新建分区 /  空着（代表剩余都给它）

9.点击KDUMP，取消Enable kdump

10.点击network & host name，点击configure，勾选自动联网
	切换到ipv4 setting，填写IP地址、子网掩码、网关
	Address      Netmask      Gateway
	10.0.0.201    24          10.0.0.2
	再填写DNS server:223.5.5.5

11.设置以上步骤后，点击root password，设置密码

```
