
1、关机重启命令

```bash
1.关机
	shutdown   关机 默认1分钟
		-h
		shutdown -h 5    5分钟后关机
		shutdown -h now  立刻关机
	
	poweroff   关机
	
	init 0     关机


2.重启
	shutdown -r 重启
		-r
		shutdown -r 5    5分钟后重启
		shutdown -r now  立刻重启
	
	reboot      重启
	
	init 6      重启
```
