
1、关机重启命令

```bash
1.关机
	shutdown   关机 默认1分钟
		-h
		shutdown -h 5    5分钟后关机
		shutdown -h now  立刻关机
	
	poweroff   关机
	
	init 0     关机


2.重启
	shutdown -r 重启
		-r
		shutdown -r 5    5分钟后重启
		shutdown -r now  立刻重启
	
	reboot      重启
	
	init 6      重启
```

2、基础命令

```bash
1.pwd 查看当前路径

[root@breadbomb ~]# pwd
/root

2.cd 切换目录
	a.回家
		cd ~
		cd /root
		cd 直接回车
	b.回到上一次所在目录
		cd -
	c.上一级目录
		cd ..

3.ls 查看文件
	-l 查看详细信息
	-a 查看所有文件，包括隐藏文件 （.file）
	
	可以使用alias命令查看别名
	ll

```
