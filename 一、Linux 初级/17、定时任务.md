
1、定时任务配置方式

```bash
1.修改系统配置文件
	/etc/crontab
	
	a.配置方法
		使用vim编辑/etc/crontab
	b.查看定时任务
		cat /etc/crontab

2.用户的定时任务
	/var/spool/cron/root #root定时任务配置文件
	/var/spool/cron/bob  #bob定时任务配置文件    普通用户很少用
	
	a.使用命令
		crontab -e  #相当于vim /var/spool/cron/currentUser  编辑当前用户定时任务配置文件
	b.查看定时任务
		crontab -l  #也可以cat /var/spool/cron/currentUser

3.查看定时任务服务运行状态
	systemctl status crond
```
