

> 核心原则：Vim 所有快捷键默认在「普通模式」下使用（按
> 
> `Esc`
> 
> 退出插入 / 可视模式切换）

## 一、基础操作（模式切换 + 保存退出）

| 快捷键          | 功能描述               | 备注          |
| ------------ | ------------------ | ----------- |
| `i`          | 光标前插入              | 进入插入模式      |
| `a`          | 光标后插入              | 进入插入模式      |
| `I`          | 行首插入               | 进入插入模式      |
| `A`          | 行尾插入               | 进入插入模式      |
| `o`          | 下一行新开一行插入          | 进入插入模式      |
| `O`          | 上一行新开一行插入          | 进入插入模式      |
| `Esc`        | 退出插入 / 可视模式 → 普通模式 | 万能返回键       |
| `:w`         | 保存文件               | 冒号开头为命令模式指令 |
| `:q`         | 退出 Vim             | 未修改时可用      |
| `:wq` / `:x` | 保存并退出              | 两种等价写法      |
| `:q!`        | 强制不保存退出            | 放弃所有修改      |
| `:wq!`       | 强制保存退出             | 权限不足时可用     |
| `ZZ`         | 普通模式：快速保存并退出       | 无需输入冒号，最便捷  |

## 二、光标移动（高效定位）

|快捷键|功能描述|扩展用法|
|---|---|---|
|`h` / `j` / `k` / `l`|左 / 下 / 上 / 右移动光标|替代方向键，效率更高|
|`w`|跳到下一个单词开头||
|`b`|跳到上一个单词开头||
|`e`|跳到当前单词结尾||
|`0`（数字零）|跳到行首|快速定位行开头|
|`$`|跳到行尾|快速定位行结尾|
|`gg`|跳到文件第一行||
|`G`|跳到文件最后一行||
|`nG`|跳到第 n 行（如 `10G` → 第 10 行）|n 为任意数字|
|`Ctrl + f`|向下翻页（forward）||
|`Ctrl + b`|向上翻页（backward）||
|`Ctrl + e`|向下滚动一行|不移动光标|
|`Ctrl + y`|向上滚动一行|不移动光标|

## 三、编辑操作（删除 + 复制 + 粘贴 + 修改）

### 3.1 删除

|快捷键|功能描述|备注|
|---|---|---|
|`x`|删除光标所在字符|单个字符删除|
|`dd`|删除整行|常用！删除后可粘贴|
|`dw`|删除当前单词（从光标到单词结尾）||
|`d0`|删除光标到行首的内容||
|`d$`|删除光标到行尾的内容||
|`D`|等同于 `d$`（删除光标到行尾）|简写更高效|
|`ndd`|删除连续 n 行（如 `3dd` → 删除 3 行）|n 为任意数字|

### 3.2 复制 & 粘贴

|快捷键|功能描述|备注|
|---|---|---|
|`yy`|复制整行|常用！复制后可粘贴多次|
|`yw`|复制当前单词||
|`y0`|复制光标到行首的内容||
|`y$`|复制光标到行尾的内容||
|`p`|在光标下一行 / 后面粘贴|小写 p 向下 / 向后|
|`P`|在光标上一行 / 前面粘贴|大写 P 向上 / 向前|
|`nyy`|复制连续 n 行（如 `5yy` → 复制 5 行）|n 为任意数字|

### 3.3 修改 & 替换

|快捷键|功能描述|备注|
|---|---|---|
|`r + 字符`|替换光标所在的一个字符|无需进入插入模式|
|`cw`|删除当前单词并进入插入模式|快速修改单词|
|`cc`|清空整行并进入插入模式|重新编辑整行|
|`c0`|删除光标到行首并进入插入模式||
|`c$`|删除光标到行尾并进入插入模式||
|`s`|删除光标所在字符并进入插入模式|单个字符修改|
|`S`|等同于 `cc`（清空整行并插入）|简写更高效|

## 四、撤销 & 重做（容错操作）

|快捷键|功能描述|备注|
|---|---|---|
|`u`|撤销上一步操作|多次按可连续撤销|
|`Ctrl + r`|重做（恢复撤销的操作）|撤销多了之后用|
|`:u!` / `:undo!`|一键还原到最后一次保存的状态|放弃所有未保存修改|

## 五、查找与替换（批量操作）

### 5.1 查找

|快捷键|功能描述|备注|
|---|---|---|
|`/关键词`|向下搜索关键词（按回车确认）|如 `/vim` 搜索 "vim"|
|`?关键词`|向上搜索关键词|与 `/` 方向相反|
|`n`|跳到下一个匹配结果|按搜索方向继续|
|`N`|跳到上一个匹配结果|与搜索方向相反|
|`:noh`|取消搜索高亮|保留匹配结果|

### 5.2 批量替换

|命令|功能描述|备注|
|---|---|---|
|`:%s/旧词/新词/g`|全文替换所有「旧词」为「新词」|`%`= 全文，`g`= 全局匹配|
|`:%s/旧词/新词/gc`|全文替换并逐一确认|`c`=confirm（确认）|
|`:n,m s/旧词/新词/g`|替换第 n 到 m 行的「旧词」为「新词」|如 `:10,20 s/abc/def/g`|

## 六、可视模式（批量选中）

|快捷键|功能描述|后续操作（选中后）|
|---|---|---|
|`v`|字符级选中（按光标移动键扩展选中范围）|`d`= 删除、`y`= 复制、`>`= 缩进|
|`V`（大写）|行级选中（整行选中，按上下键扩展）|同上|
|`Ctrl + v`|块级选中（列编辑，按方向键扩展）|适合批量添加 / 删除前缀|
|`Esc`|退出可视模式||

## 七、缩进与格式

|快捷键|功能描述|备注|
|---|---|---|
|`>>`|当前行向右缩进（默认 4 个空格）||
|`<<`|当前行向左缩进||
|`n>>`|连续 n 行向右缩进（如 `3>>` → 3 行缩进）|n 为任意数字|
|`n<<`|连续 n 行向左缩进|n 为任意数字|
|`=G`|从光标处格式化到文件末尾|自动调整缩进|
|`gg=G`|全文自动格式化缩进|代码文件常用|

## 八、实用技巧（提升效率）

1. **重复操作**：普通模式下按 `.`（点号），重复上一次编辑操作（如连续 `dd` 删除多行后，按 `.` 可快速重复删除）
    
2. **快速换行**：普通模式下按 `Shift + Enter`（部分终端支持），或 `o` 后按 `Esc`
    
3. **光标回到上次位置**：打开文件后按 `Ctrl + o`，回到上一次编辑的位置
    
4. **临时保存**：`:w 文件名` → 将当前文件另存为指定文件名（不覆盖原文件）
    
5. **显示行号**：`:set nu` → 显示行号；`:set nonu` → 隐藏行号（临时生效）
