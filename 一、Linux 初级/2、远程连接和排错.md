
1、远程连接工具

```bash
工具：
	xshell
```

2、连接所需数据

```bash
1.IP地址         10.0.0.200
2.端口号         22 （连接端口默认为22）
3.用户名、密码
	用户名：root
	密码：1
4.ssh协议
```

3、xshell无法连接虚拟机排查方案

```bash
1.本地连接vmnet8是否和虚拟机ip在相同网段

	当前虚拟机配置的ip和子网掩码
	ip:10.0.0.200
	子网掩码:24
	可以推出==>当前网段 10.0.0.1 - 10.0.0.254
	
	vmnet8查询方法
	a.本地搜索查看网络连接
	b.点击vmnet8,查看详细信息中ipv4地址
	c.若当前ip不在当前网段，属性中修改ipv4

2.检查ip地址是否正确

	输入ip a查看ip地址
	a.若为192.168开头则安装虚拟机时配置错误
	b.若为空，则网络管理被NetworkManager控制
	
	Linux网络管理服务:
		(1)network
		(2)NetworkManager
	
	关闭NetworkManager:
		(1)systemctl stop NetworkManager
		(2)systemctl disable NetworkManager

3.vmware中子网和子网掩码的配置
	子网：10.0.0.0
	子网掩码：255.255.255.0
```
