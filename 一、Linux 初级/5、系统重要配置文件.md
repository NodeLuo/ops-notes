
1、vim异常退出问题

```bash
vim 工作流程
	1.vim a.txt 进入视图模式
	2.系统生成一个隐藏文件 .a.txt.swp
	3.若正常保存退出，系统会将 .a.txt.swp 移动成 a.txt

vim 异常退出问题解决：
	1.保留修改后的内容
		a.输入 vim -r a.txt，后续直接回车
		b.输入 :wq 保存退出
		c.删除隐藏文件 .a.txt.swp
	
	2.不保留修改后的内容
		a.直接删除隐藏文件 .a.txt.swp


vim 异常退出原因：
	1.修改后直接关闭xshell
	2.修改后直接断电
	3.多个窗口同时编辑一个文件
	4.和同事同时编辑同一个文件
```
