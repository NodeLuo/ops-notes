
1、Linux系统重要配置文件

```bash
1./etc/rc.local  开机执行该文件中命令

	执行命令：给文件增加可执行权限
	chmod +x /etc/rc.local
	
	a.开机后只执行一次
	b.可以向文件中写入命令
		vim /etc/rc.local

2./etc/inittab 系统的运行级别

	a.查看系统的运行级别
		runlevel #直接回车
	
	b.系统的运行级别：
		0    关机
		1    单用户,忘记密码可以从这里找回
		2    多用户，不支持NFS 网络文件系统
		3    多用户，当前使用的命令行界面
		4    保留待开发
		5    图形界面 需提前安装
		6    重启
	
	c.进入对应级别：
		init 0 关机
		init 5 进入图形界面
		init 6 重启
	
	d.系统初始化配置：
		（1）关闭NetworkManager
			systemctl stop NetworkManager
			systemctl disable NetworkManager
		（2）创建快照

3./etc/motd 开机屏幕自动显示内容

	#
	#         ┌─┐       ┌─┐
	#      ┌──┘ ┴───────┘ ┴──┐
	#      │                 │
	#      │       ───       │
	#      │  ─┬┘       └┬─  │
	#      │                 │
	#      │       ─┴─       │
	#      │                 │
	#      └───┐         ┌───┘
	#          │         │
	#          │         │
	#          │         │
	#          │         └──────────────┐
	#          │                        │
	#          │                        ├─┐
	#          │                        ┌─┘
	#          │                        │
	#          └─┐  ┐  ┌───────┬──┐  ┌──┘
	#            │ ─┤ ─┤       │ ─┤ ─┤
	#            └──┴──┘       └──┴──┘
	#                神兽保佑
	#                代码无BUG!
	
	vim 块模式快捷键
		a.crtl+v   块模式，批量选中
		b.shift+v  批量行选中

4./etc/issue 和 /etc/issue.net  显示版本等信息

	直接清空内容：
		> /etc/issue
		> /etc/issue.net
```
